### ⚙️ PID Control Interface

A real-time graphical interface built in LabWindows/CVI with MATLAB engine integration for PID tuning, simulation and process visualization.

```
[Signal Flow]

setpoint ──▶ [ PID Controller ] ──▶ plant
                    ▲                  │
             tuning GUI         output signal
                    │                  │
            MATLAB engine ◀── feedback loop
```

**Features:**
- Live P / I / D parameter adjustment
- Real-time step response visualization
- MATLAB-powered simulation backend
- Error & control signal monitoring
